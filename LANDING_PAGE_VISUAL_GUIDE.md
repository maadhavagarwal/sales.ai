## Animation Choreography

### 🟢 **Entry Animations (On Page Load)**
```
Timeline (in sequence):
  0.0s → Badge fades in + bounces (0s delay)
  0.2s → Hero title animates in (0.2s delay)  
  0.4s → Subtitle appears (0.4s delay)
  0.6s → CTA buttons appear + hover-ready (0.6s delay)
  0.8s → Tech pills stagger (0.8s + 0.06s per item)
  1.0s → Product mockup scales in (1.0s delay)
```

### 🟢 **Continuous Animations**
```
Always Running:
  • Gradient orbs float smoothly (7-10s cycle)
  • Badge dot pulses (2s cycle)
  • Button arrow bounces on hover (2s cycle)
  • Learn More arrows wiggle (2s cycle)
```

### 🟢 **Hover/Interaction Animations**
```
On Button Hover:
  • Scale 1.0 → 1.02
  • Gradient shift: ___ (smooth transition)
  • Shadow enhances (more glow)

On Feature Card Hover:
  • Border glows: opacity 0 → 1 (500ms)
  • Icon scales: 1.0 → 1.1
  • Learn More arrow appears + animates

On Stat Card Hover:
  • Border color changes
  • Icon glow intensifies
  • Background color shifts (subtle)
```
